The task

You need to develop and install your own terminal command (script or program) on the system, which takes the path to the file directory and prints the checksum of the largest and smallest file to the screen, if possible.

Advice : use "ls",

Input data format: Path to the directory where the files will be analyzed. Output format: md5 sum and name of the largest and smallest file in the specified directory
