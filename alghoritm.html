Formal description

The following is a formal description of the algorithm:
Data: Path to directory
Result: md5 sum and names of the largest and smallest file in the directory.

  1. Sort the files in this directory and write them to an array;
  2. Print the sum for the first file in the md5 array;
  3. Print the name of the first file
  4. Print the sum for the last file in the md5 array;
  5. Print the sum for the last file in the md5 array;
